Diadem has first-party translations for English and German. All further translations are community-contributed.

:::note{icon="heart"}
Machine translations are not accepted. All translations have to be made by humans.
:::

## Using Fink

Fink is an easy to use web UI that allows you to easily view, edit and submit your translations.

<Steps>
	1. [Open Fink in your browser](https://fink.inlang.com/github.com/ccev/diadem)

	2. Log in using your GitHub account

	3. Edit and add your translations. You can add a language in the settings

	4. When done, follow the steps at the bottom of the screen to submit the translations
</Steps>
