## Features

<CardGrid>
	<Card title="Modern Map">
		Mobile-first MapLibre rendering with smooth interaction even under heavy load
	</Card>

	<Card title="Fresh UX">
		Diadem's UI has been built from the ground up, with a heavy focus on providing a good user experience
	</Card>

	<Card title="First class Unown# integration">
		Built on top of Golbat, Dragonite and Koji
	</Card>

    <Card title="Permissions">
		Supports Authentication with Discord and configurable per-area permissions based on server membership and roles
    </Card>

    <Card title="Easy to learn filters">
    	A filter system that guides the user. Easy set up for beginners, advanced options for experts
    </Card>

    <Card title="Search">
		One search bar to search through everything on the map
    </Card>

    <Card title="Tools">
		Integrated coverage map and support for Dragonite Scout
    </Card>

    <Card title="Customizable and extendible">
		Admins, as well as users, can configure Diadem to their needs, while not being overwhelmed. Diadem can also be extended to become a complete website
    </Card>

</CardGrid>
