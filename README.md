# Diadem

🌠 Next-gen map frontend for Pokémon GO

- [Docs](https://diadem.malt.ee) for feature overview and setup instructions
- [Demo](https://beta.demap.co/map) to check it out right now
- [Discord](https://discord.com/invite/VGgsQN2hYG) for support and discussions

![](https://raw.githubusercontent.com/ccev/diadem/refs/heads/main/.github/readme.png)

## Contributing

If you want to contribute anything, feel free to fork this repo and open a pull request. PRs from LLMs are okay, 
but only when the scope is manageable.

To set up a local dev environment, follow the setup instructions and run `pnpm run dev`.
A local dev server will spin up, changes can be viewed live in your browser.
